# Class iDataMatrixDetails

`iDataMatrixDetails` is one of the [`detailedResult`](auxiliary-iTextResult.html#detailedresult) in class `iTextResult`. It stores the DataMatrix code details.

>- Swift
>
>1. 
```objc

```
2. 
