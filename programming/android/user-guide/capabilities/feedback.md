# Add Scan Feedback

Trigger a beep sound or vibration when a barcode is scanned successfully.

<div class="sample-code-prefix"></div>
>- Java
>- Kotlin
>
>1. 
```java
Feedback.beep();
Feedback.vibrate();
```
2. 
```kotlin
Feedback.beep()
Feedback.vibrate()
```
